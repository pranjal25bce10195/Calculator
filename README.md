# Calculator

## Project Overview
This project is a command-line calculator built using Python. It allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. The calculator also supports continuous calculations using previous results.


## Objective
The objective of this project is to apply basic programming concepts like functions, loops, and conditionals to build a practical calculator.


## Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Continuous calculations (reuse previous result)
- Error handling for division by zero
- Input validation for operations


## Technologies Used
- Python (No external libraries required)


## How to Run the Project
1. Install Python (if not already installed)
2. Download this repository
3. Open the python file from downloads using python


## Challenges Faced
- Handling the errors made when division takes place by zero 
- Validating user input for correct operations
- Managing continuous calculations


## Learning Outcomes
- Understanding functions and dictionaries in Python
- Implementing loops and conditional statements
- Handling user input and errors
- Writing code that can be broken into small pieces and can be reused multiple times.
